# 🌐 Language Translation Tool

A simple desktop application that translates text between multiple languages in real time. Built as part of my **Artificial Intelligence Internship at CodeAlpha**.

## 📌 About the Project
This tool lets you enter text, choose a source and target language, and instantly get the translated output. It also supports copying the result and converting it to speech.

## ✨ Features
- Translate text between multiple languages
- Auto-detect source language
- Copy translated text with one click
- Text-to-speech for the translated output
- Simple and clean GUI

## 🛠️ Tech Stack
- Python
- Tkinter (GUI)
- deep-translator (Google Translate)
- gTTS (Text-to-Speech)

## 🚀 How to Run
```bash
git clone https://github.com/your-username/CodeAlpha_LanguageTranslationTool.git
cd CodeAlpha_LanguageTranslationTool
pip install -r requirements.txt
python translator.py
```
